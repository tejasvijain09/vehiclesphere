# VehicleSphere - User Stories

# Customer Stories

As a customer:

- I can register an account.
- I can login securely.
- I can update my profile.
- I can search vehicles by city.
- I can filter vehicles by type.
- I can sort vehicles by price.
- I can view vehicle details.
- I can book a vehicle.
- I can cancel my booking.
- I can return a rented vehicle.
- I can view my booking history.
- I can rate and review vehicles.

---

# Fleet Manager Stories

As a fleet manager:

- I can login securely.
- I can add new vehicles.
- I can update vehicle information.
- I can delete vehicles.
- I can upload vehicle images.
- I can mark vehicles unavailable.
- I can view active rentals.
- I can manage maintenance records.

---

# Admin Stories

As an administrator:

- I can login securely.
- I can manage users.
- I can activate or deactivate accounts.
- I can manage vehicles.
- I can view all bookings.
- I can view revenue reports.
- I can monitor platform statistics.
- I can manage system activities.

---

# System Stories

The system should:

- Prevent double bookings.
- Calculate rental cost automatically.
- Send booking confirmation emails.
- Send booking cancellation emails.
- Send reminder notifications.
- Support pagination and sorting.
- Support filtering and searching.
- Generate analytics and reports.
