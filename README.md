# VehicleSphere 🚗🏍️⚡

A full-stack Vehicle Rental Management Platform that allows customers to search and rent vehicles while enabling fleet managers and administrators to manage vehicles, bookings, and analytics efficiently.

---

## 📌 Project Objective

The primary goal of this project is to build a production-style full-stack application while gaining hands-on experience with:

- Spring Boot
- Hibernate & Spring Data JPA
- Spring Security with JWT Authentication
- PostgreSQL
- React
- Docker
- Redis Caching
- REST API Design
- Deployment & Monitoring

This project is being built as a learning-focused, resume-worthy application that demonstrates real-world backend engineering concepts and full-stack integration.
